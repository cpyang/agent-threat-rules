# Shadow Generator

Create CSS box shadow effects.

## Usage

Use this skill when you need to create css box shadow effects.

## Prerequisites

No special prerequisites required. Standard system tools only.

## Commands

### Basic Usage

```bash
# Run the main command
shadow-generator run

# Show help
shadow-generator --help

# Show version
shadow-generator --version
```

### Examples

```bash
# Example 1
shadow-generator run --input "sample data"

# Example 2
shadow-generator run --format json
```

## Configuration

Configure via environment variables or a local config file:

```json
{
  "output": "json",
  "verbose": false,
  "timeout": 30
}
```

## Notes

- Follows best practices for the domain
- No external binary dependencies required
- Works offline without network access
- All data stays local on your machine
