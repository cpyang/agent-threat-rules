# Animation Helper

Create CSS and JS animations.

## Usage

Use this skill when you need to create css and js animations.

## Prerequisites

No special prerequisites required. Standard system tools only.

## Commands

### Basic Usage

```bash
# Run the main command
animation-helper run

# Show help
animation-helper --help

# Show version
animation-helper --version
```

### Examples

```bash
# Example 1
animation-helper run --input "sample data"

# Example 2
animation-helper run --format json
```

## Configuration

Configure via environment variables or a local config file:

```json
{
  "output": "json",
  "verbose": false,
  "timeout": 30
}
```

## Notes

- Follows best practices for the domain
- No external binary dependencies required
- Works offline without network access
- All data stays local on your machine
