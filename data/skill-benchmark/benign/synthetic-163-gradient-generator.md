# Gradient Generator

Create CSS gradient backgrounds.

## Usage

Use this skill when you need to create css gradient backgrounds.

## Prerequisites

No special prerequisites required. Standard system tools only.

## Commands

### Basic Usage

```bash
# Run the main command
gradient-generator run

# Show help
gradient-generator --help

# Show version
gradient-generator --version
```

### Examples

```bash
# Example 1
gradient-generator run --input "sample data"

# Example 2
gradient-generator run --format json
```

## Configuration

Configure via environment variables or a local config file:

```json
{
  "output": "json",
  "verbose": false,
  "timeout": 30
}
```

## Notes

- Follows best practices for the domain
- No external binary dependencies required
- Works offline without network access
- All data stays local on your machine
