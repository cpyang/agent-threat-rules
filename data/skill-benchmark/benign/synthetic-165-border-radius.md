# Border Radius

Preview CSS border radius values.

## Usage

Use this skill when you need to preview css border radius values.

## Prerequisites

No special prerequisites required. Standard system tools only.

## Commands

### Basic Usage

```bash
# Run the main command
border-radius run

# Show help
border-radius --help

# Show version
border-radius --version
```

### Examples

```bash
# Example 1
border-radius run --input "sample data"

# Example 2
border-radius run --format json
```

## Configuration

Configure via environment variables or a local config file:

```json
{
  "output": "json",
  "verbose": false,
  "timeout": 30
}
```

## Notes

- Follows best practices for the domain
- No external binary dependencies required
- Works offline without network access
- All data stays local on your machine
