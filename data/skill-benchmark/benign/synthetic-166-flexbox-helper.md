# Flexbox Helper

Visual flexbox layout builder.

## Usage

Use this skill when you need to visual flexbox layout builder.

## Prerequisites

No special prerequisites required. Standard system tools only.

## Commands

### Basic Usage

```bash
# Run the main command
flexbox-helper run

# Show help
flexbox-helper --help

# Show version
flexbox-helper --version
```

### Examples

```bash
# Example 1
flexbox-helper run --input "sample data"

# Example 2
flexbox-helper run --format json
```

## Configuration

Configure via environment variables or a local config file:

```json
{
  "output": "json",
  "verbose": false,
  "timeout": 30
}
```

## Notes

- Follows best practices for the domain
- No external binary dependencies required
- Works offline without network access
- All data stays local on your machine
